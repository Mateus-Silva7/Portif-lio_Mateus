Consulta CNPJ

Aplicação web desenvolvida com HTML, CSS e JavaScript que permite consultar informações de empresas brasileiras a partir do CNPJ, utilizando a BrasilAPI. Possui interface responsiva e funcionalidade de exportação dos dados obtidos para PDF e Excel, usando as bibliotecas jsPDF e SheetJS.

To-Do List

Aplicação web responsiva para gerenciamento de tarefas (To-Do List), desenvolvida com HTML, CSS e JavaScript puro. Permite adicionar, marcar como concluída e excluir tarefas, com registro automático da data e hora de criação e conclusão. Os dados são armazenados localmente via LocalStorage, garantindo que as tarefas persistam mesmo após atualizar a página. O projeto também conta com uma interface moderna, efeitos visuais e responsividade para dispositivos móveis.
